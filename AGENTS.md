AGENTS.md
=========

Panel Color Matcher is a GNOME Shell extension to dynamically change the
top panel background color to match the color of a maximized window. The
main implementation is in file `extension.js`.

## General instructions

* We're target GNOME 49 and later
* We're targeting Wayland only

## JavaScript

* Indent with four spaces
* Use double-quotes for strings
* Use private underscore-prefixed members
